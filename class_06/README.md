## Relational Operators

-   grater than: >
-   less than: <
-   equal to: ==
-   not equal to: !=
-   grater than or equal to: >=
-   less than or equal to: <=

## Logical Operator

-   And: &&
    -   true && true -> true
    -   true && false -> false
    -   false && true -> false
    -   false && false -> false
-   Or: ||
    -   true || true -> true
    -   true || false -> true
    -   false || true -> true
    -   false || false -> false
-   Not: !
    -   true -> false
    -   false -> true
